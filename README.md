# 多人在线德州扑克（Texas Hold'em Poker）

> 技术栈：Node.js + Express + Socket.io（后端实时通信） / 原生 HTML + CSS + JavaScript（前端）

## 功能概览
- 多人在线实时对战：Socket.io（WebSocket）
- 房间系统：大厅房间列表、创建房间、加入房间
- 德扑流程：翻牌前 / 翻牌 / 转牌 / 河牌 / 摊牌
- 盲注：小盲 / 大盲（默认 10/20）
- 操作：下注 / 加注 / 跟注 / 过牌 / 弃牌 / 全押
- 牌型判断：支持从 7 张牌中选最优 5 张（含 皇家同花顺 / 同花顺 / 四条 / 葫芦 / 同花 / 顺子 / 三条 / 两对 / 一对 / 高牌）

## 本地运行

```bash
cd poker-game
npm install
npm start
```

然后访问：
- http://localhost:3000

## 使用说明（快速上手）
1. 打开页面后在「大厅」输入昵称
2. 创建房间（或加入房间列表中的房间）
3. 房主点击「开始一手」
4. 轮到你行动时，在底部操作栏进行：
   - 过牌 / 跟注 / 弃牌 / 全押
   - 下注 / 加注：在“加注到”输入框中调节数值，然后点击对应按钮

## 小提示
- 可在同一浏览器打开多个标签页模拟多人
- 掉线/离开房间会被自动视为弃牌（已投入筹码仍会进入底池）

## 目录结构
- `server.js`：服务端入口（Express + Socket.io）
- `game/`：游戏引擎
  - `pokerEngine.js`：状态机、下注轮、阶段推进、摊牌与分彩池（含 side pot 简化实现）
  - `handEval.js`：牌型评估（5 张、7 张取最优）
  - `cards.js`：牌组与洗牌工具
- `public/`：静态前端
  - `index.html` / `styles.css` / `app.js`
